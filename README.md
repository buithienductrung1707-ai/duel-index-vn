# Duel Index VN

Tra cứu thông tin thẻ bài Yu-Gi-Oh! dành cho người chơi Việt Nam, với giao diện mobile-first lấy cảm hứng từ phong cách Duel Monsters.

## Tính năng

- Tìm kiếm và xem chi tiết thẻ bài
- Thông tin bộ phát hành, độ hiếm và giá tham khảo
- Ruling và liên kết dữ liệu chính thức
- Decklist top 10, phân loại OCG/TCG
- Thống kê tỷ lệ sử dụng và tỷ lệ thắng
- Banlist OCG và TCG

Nguồn dữ liệu thẻ bài và decklist: [YGOPRODeck](https://ygoprodeck.com/).

## Chạy trên máy

Yêu cầu Node.js 22.13 trở lên.

```bash
npm install
npm run dev
```

Mở [http://localhost:3000](http://localhost:3000).

## Kiểm tra bản production

```bash
npm test
```
